# ID Generator Engine

## Purpose

Generates unique IDs automatically.

Supports configurable formats for each school.

---

## Supported IDs

Student ID

Teacher Employee ID

Admission Number

Receipt Number

Exam Number

Certificate Number

Invoice Number

Library Card Number

---

## Configuration

Each school can configure:

Prefix

Academic Year

Running Number

Padding

Separator

---
