# NR LMS Database Documentation

## Overview

The NR LMS database is designed using a modular, domain-driven architecture.

Each domain has its own documentation file containing:

* Purpose
* Tables
* Relationships
* Business Rules
* Future Enhancements

---

## Database Domains

### 01 - Authentication

Authentication, Authorization, Sessions and Security.

### 02 - School

Schools, Academic Years, Classes, Sections and Subjects.

### 03 - People

Teachers, Students, Parents and Relationships.

### 04 - Academic

Attendance, Timetable and Announcements.

### 05 - Question Bank

Question Categories, Questions, Options and Tags.

### 06 - Examination

Exams, Attempts, Answers and Results.

### 07 - Reports

Report Cards and Analytics.

### 08 - Notifications

Email, WhatsApp and Push Notifications.

### 09 - AI

AI Question Generation, AI Reports and Recommendations.

### 10 - System

Settings, File Uploads, Activity Logs and Audit.
