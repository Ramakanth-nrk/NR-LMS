# API Specifications

This directory contains the API specs for the NR LMS platform, organized by domain/entity.

## Endpoints Structure

- [Authentication](file:///c:/wamp64/www/NR-LMS/docs/03-API/Authentication.md) - Login, JWT tokens, logout, password resets.
- [School](file:///c:/wamp64/www/NR-LMS/docs/03-API/School.md) - School management, tenants, batch creation.
- [Teacher](file:///c:/wamp64/www/NR-LMS/docs/03-API/Teacher.md) - Teacher setup, profiles, assignment.
- [Student](file:///c:/wamp64/www/NR-LMS/docs/03-API/Student.md) - Student management, bulk CSV imports, enrollment.
- [Parent](file:///c:/wamp64/www/NR-LMS/docs/03-API/Parent.md) - Parent portal APIs, notifications, progress reports.
- [QuestionBank](file:///c:/wamp64/www/NR-LMS/docs/03-API/QuestionBank.md) - Question bank CRUD, categories, chapters.
- [Exam](file:///c:/wamp64/www/NR-LMS/docs/03-API/Exam.md) - Exam creation, scheduling, session autosaves, anti-cheat events.
- [Results](file:///c:/wamp64/www/NR-LMS/docs/03-API/Results.md) - Manual and auto-grading endpoints, class analytics, rankings.
