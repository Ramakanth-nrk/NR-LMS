# API Specifications: Parent Portal

## Endpoints

### 1. Get Student Performance Summary
* **Method**: `GET`
* **Route**: `/api/parent/student/:id/summary`
* **Access**: Parent
* **Description**: Retrieve overall analytics and performance metrics for a parent's child.

### 2. Get Notifications & Announcements
* **Method**: `GET`
* **Route**: `/api/parent/announcements`
* **Access**: Parent
* **Description**: Retrieve notifications, homework assignments, and school broadcasts.
