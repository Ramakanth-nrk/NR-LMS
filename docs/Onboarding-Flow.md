# School Onboarding Flow

## Objective

Define how a new school is registered and configured in NR LMS.

---

## Step 1 - Super Admin Creates School

Input

* School Name
* School Code
* Email
* Phone
* Address

Output

* School Record Created

---

## Step 2 - Configure School

* Academic Year
* Time Zone
* Language
* Currency
* Theme
* Logo

---

## Step 3 - Configure ID Patterns

Examples

Student ID

{SCHOOL_CODE}-{YEAR}-{SEQUENCE}

Teacher ID

EMP-{YEAR}-{SEQUENCE}

Admission Number

ADM-{YEAR}-{SEQUENCE}

---

## Step 4 - Enable Modules

School can enable or disable:

* Parent Portal
* Student Portal
* Teacher Portal
* Attendance
* Online Exams
* Notifications
* AI Features

---

## Step 5 - Create School Admin

System creates the first School Admin.

Credentials are shared securely.

---

## Step 6 - School Setup

School Admin creates:

* Classes
* Sections
* Subjects
* Teachers
* Students
* Parents

School is now ready to use.
