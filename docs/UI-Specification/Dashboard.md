# UI Specification: Dashboards

## View Variations by Role

### 1. Super Admin Dashboard
- **Layout**: Sidebar navigation + broad metric view.
- **Widgets**:
  - Schools list table (Status, active counts, metrics).
  - Platform audit logs viewer.

### 2. School Admin Dashboard
- **Layout**: Multi-column responsive layout.
- **Widgets**:
  - Registered teachers card grid.
  - Class batch list panels.
  - Interactive academic calendar.

### 3. Teacher Dashboard
- **Layout**: Grid layout representing class assignments.
- **Widgets**:
  - Assignment cards (Standard + Subject).
  - Pending grading queue alerts.
  - Chapter coverage / analytics graphs.

### 4. Student Dashboard
- **Layout**: Highly simplified, single-column mobile-first design.
- **Widgets**:
  - Upcoming exams timer cards.
  - Subject performance radar charts.
  - Weekly engagement streak counters.
