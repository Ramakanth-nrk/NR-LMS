# UI Specification: Student Directory & Management

## Admin / Teacher Views

### 1. Student Roster Listing
- **Controls**: Filter by standard, search bar (filter by roll number or name).
- **Actions**: Add Single Student button, Bulk Import CSV button.

### 2. Add/Edit Student Modal
- **Fields**: Full Name, Roll Number, Standard, Section, Date of Birth.
- **Validation**: Real-time roll number availability check.
