# Business Rules: Teacher Management

1. **Multiple Assignments**: A single teacher can be assigned to teach multiple class batches and multiple subjects simultaneously.
2. **Resource Isolation**: Teachers can only view student lists, view results, and create exams for classes and subjects to which they are explicitly assigned.
3. **Question Bank Ownership**: Questions added to the bank are owned by the creating teacher but are queryable/usable by other teachers within the same school tenant.
